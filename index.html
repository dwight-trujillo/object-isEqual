Object.isEqual()

Deep structural equality polyfill for ECMAScript. Compare any JavaScript values by their structure, not by reference.

v5.1.0 TC39 Stage 0 MIT License

Quick Start

npm install object-is-equal

const isEqual = require('object-is-equal');
console.log(Object.isEqual({a:1, b:{c:2}}, {a:1, b:{c:2}})); // true
console.log(Object.isEqual(0, -0, { strict: true }));          // false
